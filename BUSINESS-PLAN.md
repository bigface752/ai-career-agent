## 一句话说清楚

**帮职场人回答两个问题："我值多少钱？我该不该跳槽？"——用 AI 多角色圆桌讨论的方式，给出比职业辅导老师更客观、比 ChatGPT 更专业的个性化建议。**

---

## 一、我们要解决什么问题？

### 谁有这个问题？

3年以上工作经验、年薪15-25万、在B2B企业软件/互联网中厂/SaaS行业工作、**还没辞职但在想**的职场人。

这类人有个共同特点：**焦虑，但不敢动。**

### 他们现在怎么解决？

| 方案 | 问题 |
|------|------|
| **找职业辅导老师** | 客单价3000-10000元，水平参差不齐，同一个方法论面对所有人 |
| **问 ChatGPT / 豆包** | 没有用户画像、没有完整流程、聊完就忘、信息不专业 |
| **看小红书/知乎** | 碎片化信息，无法个性化，越看越焦虑 |
| **问朋友/家人** | 不专业，主观判断，不了解市场行情 |

### 痛点总结

1. **信息不对称**：不知道自己在市场上值多少钱
2. **决策焦虑**：想走但不敢走，怕走错、怕降薪
3. **行动盲区**：即使决定跳，不知道简历怎么改、面试怎么准备

**核心矛盾：有需求，但没有一个"专业、个性化、买得起"的解决方案。**

## 二、产品是什么？

### 用户体验流程

```
用户上传简历（30秒）
    ↓
AI 解析 + 快速诊断（60秒）
→ 立刻看到：竞争力评分、薪资定位、优劣势
    ↓
AI 引导对话（8-10分钟）
→ 像跟职业规划师聊天一样，回答几个问题
    ↓
AI 生成职业画像（30秒）
→ 完整的"AI对你的理解"卡片
    ↓
圆桌讨论（3-5分钟）
→ 6+1个AI角色（心理学家+行业总监+职业导师+AI效能专家+猎头+面试辅导，主持+1）
→ 像真的开会讨论一样，各抒己见，最后给出结论
    ↓
输出结论 + 报告
→ "建议跳槽" / "建议观望" / "建议留下" + 个性化理由
→ 如果建议留下 → 进入"当前工作辅导"模块
→ 如果建议跳 → 进入"岗位匹配"和"面试辅导"模块
```

### 四个核心模块

| 模块 | 做什么 | 用户耗时 | 价值 |
|------|--------|---------|------|
| **模块一：职业认知** | 帮用户认清自己在市场中的位置 | 15分钟 | 从"模糊焦虑"到"清晰定位" |
| **模块零：当前工作辅导** | 如果不该跳，告诉你怎么在当前岗位提升 | 20分钟 | 覆盖70%"不该跳"的用户 |
| **模块二：岗位匹配** | 输入目标岗位JD，告诉你匹配度和差距 | 5分钟 | 知道"这个岗位我能不能去" |
| **模块三：面试辅导** | 模拟面试 + 答案优化 | 20分钟 | 高仿真模拟面试训练 |

### 核心差异化：圆桌讨论

**这是区别于所有竞品的关键。**

市面上所有 AI 产品（ChatGPT、豆包、Kimi）都是"一个AI跟你聊"。

我们是**多个AI角色同时讨论你的问题**：

```
┌─────────────────────────────────────────────────────────┐
│                    圆桌讨论示例                            │
├─────────────────────────────────────────────────────────┤
│                                                           │
│  【心理学家】                                              │
│  "从你的描述看，你的焦虑更多来自'同龄人比较'，              │
│   而不是真正的工作不满。建议先区分'真实需求'和'被放大的焦虑'。" │
│                                                           │
│  【行业总监】                                              │
│  "B2B SaaS行业今年确实收紧了，但你所在的细分领域             │
│   （数据类工具）还在增长，跳槽窗口期还在。"                 │
│                                                           │
│  【职业导师】                                              │
│  "你的画像显示你有跨部门协作能力，这是稀缺的。               │
│   但你缺乏团队管理经验，这是你薪资上不去的关键卡点。"       │
│                                                           │
│  【AI效能专家】                                            │
│  "你目前还没有使用AI工具的习惯，这是你的短板。               │
│   产品运营岗用AI提效的空间很大，建议优先学习。"             │
│                                                           │
│  ── 共识结论 ──                                            │
│  1. 暂时不建议跳槽，先补管理经验                            │
│  2. 优先学习AI工具，提升竞争力                              │
│  3. 6个月后重新评估                                        │
│                                                           │
│  ── 分歧点 ──                                              │
│  心理学家认为焦虑被放大，行业总监认为行业窗口期真实存在      │
│                                                           │
└─────────────────────────────────────────────────────────┘
```

**用户看到的不是"一个AI说的"，而是"多个专家讨论后的共识"——更有说服力，更有信任感。**

## 六、竞争优势

### 竞品对比

| 竞品 | 价格 | 问题 | 我们的优势 |
|------|------|------|-----------|
| **职业辅导老师** | 3000-10000元 | 水平参差、同一方法论、无法跨行业 | AI多角色讨论、跨行业个性化、价格低90% |
| **ChatGPT/豆包** | 免费-20元/月 | 无画像、无流程、无专业分析 | 持久化画像 + 结构化流程 + 多Agent协作 |
| **猎聘/简历网** | 免费 | 甲方视角、无辅导 | 用户视角 + JD打分 + 面试辅导 |

### 核心壁垒

1. **画像即壁垒**：用户投入越多（对话、反馈、修正），画像越精准，切换成本越高
2. **圆桌讨论机制**：多AI角色讨论的技术门槛和Prompt工程积累
3. **行业认知**：创始人对B2B企业软件行业的深度理解

## 七、未来商业化可能性

### 短期（6个月）：验证阶段

- 目标：100 个付费用户
- 验证：用户愿不愿意付费？画像质量够不够？
- 收入：副业收入，不指望养活自己

### 中期（1-2年）：增长阶段

| 方向 | 可行性 | 说明 |
|------|--------|------|
| **提高客单价** | ⭐⭐⭐⭐⭐ | 49元→99元，199元→399元，用户愿意为高质量付费 |
| **订阅制** | ⭐⭐⭐⭐ | 月付/年付，持续更新画像、定期职业评估 |
| **企业版** | ⭐⭐⭐⭐ | 面向企业HR，批量员工职业发展评估 |
| **转介绍** | ⭐⭐⭐⭐⭐ | 好产品自然传播，转介绍率目标≥25% |

### 长期（3-5年）：规模化

| 方向 | 可行性 | 说明 |
|------|--------|------|
| **垂直领域扩展** | ⭐⭐⭐⭐ | 从B2B软件扩展到金融、医疗、教育等行业 |
| **职业发展全链路** | ⭐⭐⭐⭐ | 从"该不该跳"扩展到"职业规划、晋升辅导、转型指导" |
| **AI职业教练** | ⭐⭐⭐⭐⭐ | 持续陪伴式职业发展顾问 |
| **培训/课程** | ⭐⭐⭐ | 基于画像推荐个性化学习路径 |

### 商业化策略（已记录，后续细化）

- 付费/免费用户记忆分层策略
- 用户画像深度体验区分
- 49元 vs 199元功能差异

## 八、风险和应对

| 风险 | 概率 | 影响 | 应对方案 |
|------|------|------|---------|
| **AI报告质量不稳定** | 高 | 致命 | Prompt工程优化 + 温度控制 + 质量checklist |
| **用户不愿意付费** | 中 | 高 | 先免费验证价值，再逐步收费 |
| **Token成本过高** | 中 | 中 | 分层模型（4o+mini），MVP后优化 |
| **竞品跟进** | 低 | 中 | 画像壁垒 + 行业认知 + 持续迭代 |
| **创始人精力不足** | 高 | 中 | 严格控制每月≤20小时，优先级管理 |

## 十、我需要你做什么？

### 角色定义

**联合创始人 / 运营合伙人**

### 具体职责

| 职责 | 说明 |
|------|------|
| **用户运营** | 种子用户获取、用户反馈收集、社群运营 |
| **内容运营** | 小红书/知乎内容创作、用户案例包装 |
| **数据分析** | 用户行为分析、转化漏斗优化 |
| **商业化** | 定价策略、付费转化、用户分层 |

### 为什么需要你？

1. **我负责产品和技术，你负责运营和增长** — 分工明确
2. **两个人比一个人更有执行力** — 副业需要互相督促
3. **你的视角能补我的盲区** — 我偏产品思维，你偏用户思维
4. **这是一件有长期价值的事** — 不是一锤子买卖，是可以持续做3-5年的事业
