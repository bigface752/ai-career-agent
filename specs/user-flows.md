```
用户首次访问
    ↓

    ↓
┌─────────┬─────────┬─────────┐
│ 入口A    │ 入口B    │ 入口C    │
│ 职业认知  │ 岗位匹配  │ 面试辅导  │
└────┬────┘ └────┬────┘ └────┬────┘
     │           │           │
     ↓           │           │
  模块一         │           │
     │           │           │
  圆桌结论       │           │
   ╱    ╲        │           │
该跳    不该跳    │           │
  ↓       ↓      │           │
模块二   模块零   │           │
  ↓       ↓      │           │

  ↓              ↓           ↓

```

## 模块一：职业认知流程

```
Step 1: 上传简历
  页面：上传页
  操作：拖拽或点击上传 PDF（≤10MB，V1仅支持PDF）
  反馈：上传进度条 → "正在分析..."
  耗时：30秒
    ↓
Step 2: 确认解析信息
  页面：解析确认页
  内容：AI提取的职位、行业、年限、技能、项目
  操作：用户修正错误 → 点击确认
  耗时：30秒
    ↓
Step 3: 快速诊断
  页面：诊断结果页
  内容：竞争力定性评级（强/中/弱）+ 薪资分位 + Top 2短板 + Top 2优势
  操作：查看 → 可截图分享 → 点击"继续深度分析"
  耗时：2-3分钟（5 Agent × 3次并行，流式输出）
  ⚡ 首次价值交付点
    ↓
Step 4: 引导式对话
  页面：对话页
  内容：AI逐轮提问（4-6轮）
  操作：用户回答 → AI追问 → ...
  中间反馈：第4轮后显示"初步发现"卡片
  耗时：8-10分钟
  ⚠️ 断点：用户可随时退出，进度保存
    ↓
Step 5: 生成职业画像
  页面：画像报告页
  内容：核心标签 + 价值排序 + 风险偏好 + 市场定位
  操作：查看画像 → 可修正不准确的标签
  耗时：30秒
    ↓
Step 6: 圆桌讨论
  页面：圆桌讨论页
  内容：6角色辩论（心理学家+行业总监+职业导师+AI效能专家+猎头+面试辅导）+ 共识结论 + 分歧点
  操作：查看辩论过程 / 折叠只看结论
  耗时：3-5分钟
    ↓
Step 7: 结论判断
  系统输出：建议（跳/观望/不跳）+ 理由
  用户选择：
    → "该跳" → 进入模块二
    → "不跳" → 进入模块零
    → "我要看机会" → 进入模块二（用户可拒绝结论）
```

## 模块零：当前工作辅导流程

```
触发：模块一结论为"不该跳" + 用户选择"接受"
    ↓
Step 1: 结论呈现
  内容：圆桌结论 + 理由 + "进入工作辅导"按钮
  操作：用户选择进入 / 拒绝进入模块二
    ↓
Step 2: 补充当前工作信息
  页面：对话页
  内容：AI基于已有画像追问（3-5轮）
  目的：补全 portrait_json.current_work
  耗时：3-5分钟
    ↓
Step 3: 圆桌讨论
  角色：职业导师 + 猎头 + 头部企业专家 + AI效能专家
  内容：4角色讨论当前工作情况
  耗时：5-8分钟
    ↓
Step 4: 输出提升方案
  内容：当前定位 + 天花板 + Top 3提升方向 + 行动计划
  格式：在线查看
    ↓
Step 5: 后续跟进
  · 3个月后自动提醒回访
  · 用户可随时回来更新画像
  · 情况变化可重新进入模块一评估
```

## 模块二：岗位匹配流程

```
入口：首页直接进入 / 模块一完成后进入
    ↓
Step 1: 输入目标岗位
  页面：JD输入页
  方式：粘贴JD文字 / 输入岗位名称
  耗时：30秒
    ↓
Step 2: AI解析JD
  内容：提取岗位要求（技能/经验/学历/薪资/地点）
  耗时：30秒
    ↓
Step 3: 匹配评分
  内容：总体匹配定性评级（强/中/弱）+ 4维度评级 + 差距 + 优势
  耗时：60秒
    ↓
Step 4: 圆桌讨论（辩论式，2轮）
  角色：岗位洞察 + 行业总监 + 猎头（3角色，无独立主持）
  Round 1: 各自亮明立场（150-250字/角色）
  Round 2: 交叉质疑 + 最终立场
  综合：MiMo提取共识/分歧，生成投递建议
  耗时：1-2分钟
    ↓
Step 5: 输出匹配报告
  内容：匹配度 + 差距分析 + 优化建议 + 圆桌结论
  操作：查看 → 可进入模块三（面试辅导）
    ↓
可选：用户输入多个JD，每个都做匹配分析
```

---

## 模块三：面试辅导流程

```
入口：首页直接进入 / 模块二完成后进入
    ↓
Step 1: 选择面试目标
  页面：面试准备页
  操作：选择目标岗位（复用模块二的JD或新输入）+ 选择面试轮次
    ↓
Step 2: 生成面试题
  方式：单Agent直接生成（非圆桌）
  内容：专业题（3-5）+ 行为题（2-3）+ 非标准题（1-2）
  耗时：60秒
    ↓
Step 3: 模拟面试
  页面：面试页
  体验：AI逐题提问 → 用户输入回答 → AI追问（最多3次/题）
  耗时：10-15分钟
  ⚠️ 断点：用户可随时退出，进度保存
    ↓
Step 4: 答案评估圆桌
  角色：面试辅导 + AI效能专家
  内容：逐题评估 + 优化建议 + 回答示范
  耗时：3-5分钟
    ↓
Step 5: 输出面试报告
  内容：整体评分 + 每题评估 + 优化建议 + 示范回答
  格式：在线查看
```

## 职业仪表盘流程

```
用户登录后默认进入
    ↓
展示内容：
  · 画像完成度（进度条 + 各维度状态）
  · 各模块最新结论摘要
  · 下一步建议（"完成XX模块可解锁更多洞察"）
  · 最近一次报告链接
    ↓
操作：
  · 点击任意模块 → 进入该模块
  · 点击画像 → 查看完整画像
  · 点击报告 → 查看历史报告
```
