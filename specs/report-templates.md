# PRD 补充：报告模板结构

> **文件职责**：报告模板。所有报告的格式定义都记录在这里。

## 模块一：职业认知报告

```
┌─────────────────────────────────────────────┐
│  你的职业认知报告                              │
│  [姓名] · [职位] · [行业]                     │
│  生成日期：2026-XX-XX                         │
├─────────────────────────────────────────────┤
│                                               │
│  1. 市场定位                                   │
│  ┌─────────────────────────────┐              │
│  │  竞争力评级：4/5良好            │              │
│  │  一句话定位：[个性化定位]      │              │
│  │  薪资定位：P55（中上）         │              │
│  │  当前市场区间：18-28万/年      │              │
│  └─────────────────────────────┘              │
│                                               │
│  2. 职业画像                                   │
│  · 核心标签：B2B产品运营 / 数据驱动 / 用户增长  │
│  · 价值排序：成长空间 > 薪资 > 工作生活平衡     │
│  · 风险偏好：中等                              │
│                                               │
│  3. 你的优势（Top 2）                          │
│  · [优势1] — 市场价值：[说明]                  │
│  · [优势2] — 市场价值：[说明]                  │
│                                               │
│  4. 你的短板（Top 2）                          │
│  · [短板1] — 影响：[说明] — 建议：[行动]       │
│  · [短板2] — 影响：[说明] — 建议：[行动]       │
│                                               │
│  5. 行动建议（Top 3）                          │
│  · [最优先] [行动] [时间线]                    │
│  · [次优先] [行动] [时间线]                    │
│  · [可选] [行动] [时间线]                      │
│                                               │
│  6. 圆桌讨论结论（6角色：心理学家+行业总监       │
│     +职业导师+AI效能专家+猎头+面试辅导）        │
│  · 共识：[2-3条]                              │
│  · 分歧：[1-2条]                              │
│  · 建议：跳/观望/不跳                          │
│                                               │
│  7. 风险提示（来自心理学家质疑者角色）           │
│  · ⚠️ [质疑点1] — [证据] — [对你的意义]        │
│  · ⚠️ [质疑点2] — [证据] — [对你的意义]        │
│  · 总结：[一句话风险提示]                       │
│  （如果心理学家判断无需质疑，此区不显示）         │
│                                               │
│  ─────────────────────────────                │
│  想深入分析？                                   │
│  [进入岗位匹配 →]                              │
│  [进入面试辅导 →]                              │
│  [进入工作辅导 →]                              │
│                                               │
│  免责声明：本报告基于AI分析，仅供参考。         │
└─────────────────────────────────────────────┘
```
